# PERKS Platform Data Product

This repository collects the public descriptors for the PERKS platform data product. Each solution keeps its own OpenAPI contract in its folder, while the repository-level [DPROD](https://www.omg.org/spec/DPROD) descriptor provides a semantic annotation and contextual information to process the content of this folder.

OpenAPI descriptors for each solution can be viewed and edited with tools such as [Swagger Editor](https://editor.swagger.io/).

## Working With The Descriptors

- Use the solution folders as the canonical source for each API contract.
- Use [dataproduct-dprod.ttl](dataproduct-dprod.ttl) when you need a semantic description of the overall data product.
